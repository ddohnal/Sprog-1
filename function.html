Funkce

Popis

  • samostatná část programovu
  • funkce umožnují psát program přehledněji a úsporněji
  • lze ji libovolně volat odkudkoliv
  • při každém volání se provedou příkazy uvnitř funkce
  • funkce může mít své vstupy (parametry funkce) i výstup (návratová hodnota)

Vytvoření funkce

  • funkce obecně
  • typVysledku = datový typ návratové hodnoty
    • známé datové typy návratových hodnot: int, string, bool, int[], ...
    • void: datový typ, který určuje, že funkce nebude nic vracet
  • jmenoFunkce = platí stejná pravidla jako pro pojmenování proměnné
  • typ1 jmeno1 = datový typ a identifikátor parametru
  • parametry se uvnitř funkce chovají jako klasické proměnné
  • pokud budeme chtít, aby funkce něco vracela, použijeme příkaz return (funkce se v tomto bodě ihned ukončí)
  • funkce budeme definovat uvnitř třídy program (mimo funkci main)

Úkol

Vytvořte funkci ValidaceHesla, která bude brát textový řetězec jako parametr. Smyslem funkce bude určit, zda-li uživatel zadal heslo splňující následující kritéria:

  1. heslo musí mít více jak 6 znaků a méně než 12,
  2. heslo musí obsahovat alespoň jedno velké písmeno,
  3. heslo musí obsahovat alespoň jednu číslici,
  4. heslo musí obsahovat alespoň jeden speciální znak !@#$%^&*()-+?_=,<>/.,
  5. heslo nesmí obsahovat dva stejné znaky po sobě,
  6. uživatel zadá heslo znova -> hesla se musí shodovat.
